## 🚀 How to Use

### Quick Start (30 seconds)
```
1. Open BatchingApp
2. Find "Print Ticket" settings panel (upper right)
3. Click 🖨️ Test Print
4. Review preview window
5. Click Close
```

### Step by Step
```
1. Open BatchingApp application
2. Look for Print Ticket panel in upper right
3. Optionally customize:
   - Header text (company name, address)
   - Paper width (default 32 characters)
   - COM port (if different)
   - Baud rate (default 9600)
4. Click 🖨️ Test Print button
5. Preview window appears showing sample ticket
6. Choose action:
   - 🖨️ Print Ticket → Send to printer
   - 📋 Copy Text → Copy to clipboard
   - Close → Just close window
```

### During Actual Weighing
```
1. Enable "Auto-print after capture" checkbox
2. Load recipe
3. Start batch
4. Weigh ingredients
5. Click Capture Now
6. Auto-prints ticket (if printer ready)
7. Move to next ingredient
8. Repeat
```
